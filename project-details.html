О проекте

Платформа обучения STEM на основе технологии цифровых двойников представляет собой инновационное решение для дистанционного обучения промышленным процессам через облачные технологии с возможностью удалённого управления реальным оборудованием.

Проблема и решение

Решаемые проблемы

  • Высокие затраты на создание лабораторий в учебных заведениях
  • Низкая доступность практического обучения для студентов
  • Отставание образовательной инфраструктуры от требований индустрии 4.0

Цель проекта

Создание образовательной платформы, использующей цифровых двойников для дистанционного обучения STEM-дисциплинам и управления физическими роботизированными системами.

Технологическое решение

Доступное обучение промышленным процессам для учреждений среднего специального, профессионального и высшего образования через облачные технологии.

Ключевые особенности:

  • Цифровые двойники промышленного оборудования
  • Удалённое управление реальными системами
  • Облачная инфраструктура для доступа
  • Интерактивные 3D-модели лабораторных установок
  • Образовательные курсы и методические материалы

Текущий статус проекта

Прототип готов

Создан прототип цифрового двойника с возможностью удалённого управления реальным оборудованием

Образовательный контент

Разработан учебный курс и 3D-модели лабораторных установок

Анализ рынка

Проведены тестирования платформы с 13 компаниями для анализа требований рынка

Планы по развитию

Исследования и доработки

Расширение функционала платформы за счёт улучшения цифрового двойника и интеграции новых промышленных сценариев.

Коммерциализация

  • Продажа образовательных услуг
  • Лицензирование платформы
  • Ориентация фокуса на индустриальные предприятия для повышения масштаба проекта

Ожидаемые результаты

Охват адресного рынка на уровне 5% или 2 млн долларов США.
